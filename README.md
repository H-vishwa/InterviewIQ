# InterviewIQ

## Project Description
InterviewIQ is a comprehensive interview platform designed to aid job seekers and recruiters in the hiring process. The platform provides tools for conducting interviews, evaluating candidates, and managing the hiring workflow efficiently.

## Features
- **User-Friendly Interface**: Intuitive navigation and layout to enhance user experience.
- **Interview Scheduling**: Schedule interviews conveniently with automated reminders.
- **Question Bank**: Extensive collection of interview questions across various domains.
- **Assessment Tools**: Built-in tools to assess candidate responses and skills.
- **Feedback System**: Collect and manage feedback from interviewers easily.

## Technology Stack
- **Frontend**: React.js
- **Backend**: Node.js with Express
- **Database**: MongoDB
- **Deployment**: Docker containers on AWS

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/H-vishwa/InterviewIQ.git
   ```
2. Install dependencies:
   ```bash
   cd InterviewIQ
   npm install
   ```
3. Set up environment variables according to `.env.example` and rename it to `.env`.
4. Run the application:
   ```bash
   npm start
   ```

## Usage Guidelines
- Navigate to the platform's URL once the application is running.
- Create an account or log in to access the features.
- Follow the on-screen instructions to schedule interviews and manage your interview process.
